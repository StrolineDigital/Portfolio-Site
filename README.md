# Portfolio Site
A ground-up build of a fully responsive site that showcases my programming projects
## Description
The goal of this project was to create a preliminary website for showcasing my programming projects. The website is designed to be able to read an about me, view recent projects, contact me, and view my resume.

![Screenshot 2023-12-11 132224](https://github.com/StrolineDigital/Portfolio-Site/assets/148625766/6d327cab-cf96-49e0-a387-41a244e2aeea)

## Usage
The user is advised that this site is fully responsive and has interactive features including clickable images which glow on hover, a sticky navbar that remains at the top of the page, and interactive contact links.
